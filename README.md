## ✍️ Descripción

Este proyecto es una aplicación web desarrollada con **Vite + TypeScript** cuyo objetivo es demostrar un sistema básico de:

- Autenticación de usuarios
- Manejo de sesiones en frontend
- Protección de rutas según rol

Se implementan dos tipos de usuario:

- **Admin**
- **Client**

Cada uno tiene acceso a diferentes secciones de la aplicación.

---

## ⚠️ Seguridad

Este proyecto es **educativo**.

NO es seguro para producción porque:

- Usa `localStorage`
- No hay backend
- Los datos pueden ser modificados desde el navegador

👉 En una aplicación real, la seguridad debe manejarse en el backend.
